# Iniciando os conceitos de Node.JS
## Criando um CRUD

```
$ yarn # Instalar as dependências
```

```
$ yarn dev # Roda a aplicação, no localhost:3333
``` 

```
$ yarn test # Roda os testes automatizados
```
